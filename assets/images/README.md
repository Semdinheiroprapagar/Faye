# Organização de Imagens

Coloque suas imagens nas subpastas dentro de `assets/images/products`.

## Categorias sugeridas

- `best-sellers`
- `lancamentos`
- `linha-360`
- `feminino`
- `masculino`
- `acessorios`

Substitua o arquivo `placeholder.png` ou adicione suas próprias imagens utilizando os nomes indicados no arquivo `script.js`.

## Logo

- Substitua `assets/images/logo.png` pela sua arte oficial mantendo o mesmo nome de arquivo.
- Use preferencialmente imagens em PNG com fundo transparente.
- O mesmo arquivo é utilizado como favicon (ícone da aba do navegador).
